## ✅ VALIDATION EXAMPLES

### Test Aadhaar Validation

```python
import re

schema = {
    "pattern": r"^\d{12}$"
}

test_cases = [
    ("123456789012", True),   # Valid
    ("12345678901", False),   # Too short
    ("AAAA56789012", False),  # Contains letters
    ("", False),              # Empty
]

for value, expected in test_cases:
    valid = bool(re.match(schema["pattern"], value))
    status = "✓" if valid == expected else "✗"
    print(f"{status} '{value}': {valid}")
```

### Test PAN Validation

```python
import re

schema = {
    "pattern": r"^[A-Z]{5}[0-9]{4}[A-Z]{1}$"
}

test_cases = [
    ("AAAPA1234F", True),     # Valid
    ("aaapa1234f", False),    # Lowercase
    ("AAAPA123F", False),     # Missing digit
    ("", False),              # Empty
]

for value, expected in test_cases:
    valid = bool(re.match(schema["pattern"], value.upper()))
    status = "✓" if valid == expected else "✗"
    print(f"{status} '{value}': {valid}")
```

## 🐛 TROUBLESHOOTING

### Issue 1: "ModuleNotFoundError: No module named 'playwright'"

**Solution:**
```bash
pip install playwright
playwright install chromium
```

### Issue 2: "Connection refused" or "Timeout"

**Solution:**
```bash
# Check internet connection
ping google.com

# Check if Udyam portal is accessible
curl -I https://udyamregistration.gov.in

# Use pre-built schema instead
cp schema/schema_udyam.json schema/schema_udyam_scraped.json
```

### Issue 3: "playwright._impl._api_types.Error: Target page, context or browser has been closed"

**Solution:**
```bash
# Browser crashed. Reinstall Playwright
playwright install chromium --with-deps

# Or use the pre-built schema
python -c "import shutil; shutil.copy('schema/schema_udyam.json', 'schema/schema_udyam_scraped.json'); print('✓ Using pre-built schema')"
```

### Issue 4: "JSONDecodeError" when reading schema

**Solution:**
```bash
# Schema file is corrupted. Redownload
curl https://raw.githubusercontent.com/... > schema/schema_udyam.json

# Or use the pre-built copy
cp schema/schema_udyam.json schema/schema_udyam_scraped.json
```
